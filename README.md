# AI-Based Intrusion Detection System

This project is an AI-powered Intrusion Detection System that detects malicious network activity using machine learning.

## Features
- Real-time packet capture using Scapy
- Machine learning-based attack detection
- Data preprocessing and feature extraction
- Interactive dashboard (Streamlit)

## Modules
- Data Preprocessing
- Model Training
- Network Data Capture
- Backend Integration
- Frontend Visualization

## Tech Stack
- Python
- Scapy
- Scikit-learn
- Streamlit

## How to Run
1. Run sniffer.py to generate live_input.csv
2. Use backend to generate predictions
3. Launch frontend dashboard
